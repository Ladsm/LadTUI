# The NumericUpDown Class

## Info

### The NumericUpDown Class is a NumericUpDown that can be used for int's that uses the arrow keys.

## How to call and Use

To use it, create an object with data type NumericUpDown.

```cpp
ladtui::NumericUpDown Numupdown;
```

Then to use it, make an int and set it to Numupdown.useNumupdown();.

```cpp
int num1 = Cupdown.useNumupdown();
```
